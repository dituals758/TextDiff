# TextDiff — Сравнение текстов

Локальный инструмент для сравнения двух текстов с подсветкой различий. Работает полностью в браузере — данные не покидают ваш компьютер.

## Возможности

- **3 режима сравнения**: по символам, по словам, по строкам
- **2 режима отображения**: единый (unified) и раздельный (side-by-side)
- **Показ невидимых символов**: пробелы `·`, табуляции `→`, переносы строк `¶`
- **Экспорт в .docx** — результат можно отправить коллеге или сохранить в архив
- **Offline** — никаких запросов к серверу, полная конфиденциальность
- **Настройки сохраняются** в localStorage между сессиями

## Использование

1. Откройте `TextDiff.html` в браузере
2. Вставьте два текста для сравнения
3. Выберите режим (символы / слова / строки) и вид (unified / split)
4. Нажмите **Сравнить тексты** или `Ctrl+Enter`
5. Экспорт в .docx — кнопка в шапке результата

## Требования

Любой современный браузер (Chrome, Firefox, Edge, Safari). Не требует установки.
